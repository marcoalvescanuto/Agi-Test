# README #

This Project is intended for Web tests of the Agi Test application.

### Technologies ###

* Java
* Cucumber
* Serenity
* JUnit

### Dependencies ###

* Java
* Maven | Grandle

### Usage ###

To run all tests

```
  mvn clean verify
```

### Usage ###

To get reports, acess

```
  ./target/site/serenity/index.html
```
